Terms of Use

 
Within the Help dropdown menu, users can select the Terms of Use option via the Legal dropdown menu to display the Terms of Use window.
 

Terms of Use

  • This window displays the CSET Terms of Use that users agree to when using the tool and a link to the open-source licenses.

OK

  • Select the OK button to exit the CSET Terms of Use window.
