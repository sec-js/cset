Resource Library - Browse

 
Selecting Resource Library in the main navigation bar displays the Resource Library window with the Search tab displaying by default. Select the Browse tab option of the Resource Library to view the document tree.
 

Filter

  • Use the Filter field to enter one or more keywords related to the desired documents, such as the topic, the authoring organization, or document type, to filter the document tree.
Top

Collapse All

  • Select the Collapse All button to view only the main categories of the document tree.
Top

Expand All

  • Select the Expand All button to view all main categories, sub-categories, and documents within the document tree.
Top

Document Tree

  • The document tree displays the following main categories (branches), each with their own sub-categories and documents (leaves):
  • Publisher
  • Reports
  • Templates
  • Standards
  • Publication Year
  • Guidance
  • Cyber Security Procurement Language
  • Note: Selecting the hyperlinks within this category will open the content in a CSET window instead of separate file.
  • Catalog of Recommendations
  • Note: Selecting the hyperlinks within this category will open the content in a CSET window instead of separate file.
  • Select the expand/collapse (/) icons to open or close a branch. Expanding a branch will display additional sub-categories (branches) and documents (leaves).
  • Users can click the hyperlinked document title to open the document.
  • Note: Documents may appear under multiple categories.
