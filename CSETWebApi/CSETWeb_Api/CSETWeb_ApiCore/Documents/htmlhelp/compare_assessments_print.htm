Compare Assessments

 
Selecting the Next button on the Comparison Details window opens the Compare Assessments page. The Compare Assessments page includes a variety graphs and data that helps the user compare and analyze performance over time or across the different areas of the organization or assets.
 
 

Comparison Data

  • The Compare Assessments page includes the following tabs along with the specified graphs and data:
  • Overall Summary
  • Overall Average Summary
  • Standards Answers
  • Maturity Compliance
  • Components Answers
  • Category Average for All Assessments
  • Commonly Missed Questions
  • Individual Assessment Comparison
  • Assessment Answer Summary
  • Overall Comparison
  • Comparison of Security Assurance Levels
  • Category Percent Comparisons
  • Best to Worst

Back

  • Select the Back button to return to the Comparison Details window.

Generate Compare Report

  • Select the Generate Compare Report button to generate a report that contains the Security Assurance Levels (SALs), the graphs, and the document library.
