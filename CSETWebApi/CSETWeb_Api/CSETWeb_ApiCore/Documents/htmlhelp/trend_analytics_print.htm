Trend Analytics

 
Selecting the Next button on the Trend Details window opens the Trend Analytics page. The Trend Analytics page includes a variety graphs that helps the user identify any trends among the selected assessments.
 

Trend Graphs

  • The Trend Analytics page includes the following graphs:
  • Overall Percentage Compliance Score
  • Top 5 Most Improved Areas
  • Top 5 Areas of Concern
  • Category Percent Comparisons

Back

  • Select the Back button to return to the Trend Details window.

Generate Trend Report

  • Select the Generate Trend Report button to generate a report that contains the Security Assurance Levels (SALs), the graphs, and the document library.
