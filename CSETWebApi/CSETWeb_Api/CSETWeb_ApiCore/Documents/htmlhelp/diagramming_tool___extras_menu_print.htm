Diagramming Tool - Extras Menu

 
Selecting Extras in the Diagram Tool Bar displays the Extras dropdown menu.
 

Language

  • Select the Language option to modify the interface language.
  • From the Language dropdown menu, users can select from a list of languages.
  • Users must refresh the diagramming tool to update the interface with the new language.

Appearance

  • Select the Appearance option to modify the appearance of the diagramming tool.
  • From the Appearance dropdown menu, users can select:
  • Light
  • Dark
  • Automatic
  • Classic
  • Selecting the Classic option updates the current diagramming tool interface to the classic interface.

Theme

  • Select the Theme option to modify the theme of the diagramming tool.
  • From the Theme dropdown menu, users can select:
  • Automatic
  • Classic
  • Simple
  • Sketch
  • Minimal
  • Atlas

Plugins

  • Select the Plugins option to open the plugins window.
  • Select the Help button to review the descriptions of the available plugins.
  • Select the Add button to add a plugin to the diagramming tool.
  • Select a plugin from the dropdown menu and select the OK button.
  • Select the Apply button to add the plugin to the diagramming tool.

Edit Diagram

  • Select the Edit Diagram option to open the mxgraph editor and edit the diagram.
  • Note: This feature is intended for developer use.

Configuration

  • Select the Configuration option to open the Configuration editor.
  • Note: This feature is intended for developer use.
