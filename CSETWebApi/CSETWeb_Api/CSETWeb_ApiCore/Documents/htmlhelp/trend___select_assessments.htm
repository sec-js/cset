Trend - Select Assessments

 
Selecting the Select Assessments button on the Trend Details window opens the Select Assessments window. Users are able to select the assessments with which to perform the Trend aggregation function.
 

Assessment Selection

  • Users must select two or more assessments to perform the Trend aggregation function.
  • Note: Users are recommended to analyze no more than five assessments within a single trend.
  • Upon selecting the first assessment, assessments incompatible with the selected assessment are disabled.
  • Note: CSET currently only supports the trend aggregation function for standard-based assessments.
Top

Assessment Compatibility

  • Upon selecting two or more assessments, the Select Assessments window displays the assessment questions and requirements compatibility scores.
  • Note: The compatibility scores reflect the similarity of the standards for the selected assessments, not the likeness of the answers given within each assessment. A low compatibility percentage indicates minimal overlap in questions or requirements within the selected assessments.
Top

OK

  • Select the OK button to confirm the assessment selections with which to perform the Trend aggregation function.
  • Upon selection, the Select Assessments window is closed and the user is returned to the Trend Details window.
