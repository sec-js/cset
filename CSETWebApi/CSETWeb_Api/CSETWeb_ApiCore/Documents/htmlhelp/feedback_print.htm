Feedback

 
Selecting Feedback within the Results menu displays the Feedback page. The Feedback page allows users to submit any feedback regarding questions/requirements to the CSET Standards Administrator.
 

Copy Text

  • Select the Copy Text button to copy the feedback text below to the clipboard.

Email

  • Select the Email button to email the feedback text below to the CSET Standards Administrator.
  • Upon selection, the user's default email application will open.
  • The email will auto-populate the subject line, the recipient address, and the body of the email with the feedback text.

Feedback Text

  • The feedback text includes a list of feedback recorded during the assessment and their associated question IDs.
  • Feedback text cannot be edited on the Feedback page.
  • To edit the feedback, visit the Feedback section of the corresponding question/requirement.
