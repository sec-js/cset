Details/Standard Section

 
Selecting the Details/Standards button on the assessment screen displays the Details/Standards section.
 

Details/Standard

  • Based on the selected assessment type and the assessment mode, the Details/Standard section may display the following information:
  • Title
  • The title text is an identifier for the question/requirement.
  • Category
  • The name of the sub-domain or sub-category associated with the question/requirement.
  • Security Assurance Level (SAL)
  • The Security Assurance Level text indicates the SAL for the question/requirement.
  • Standard Specific Requirement (Displays in Questions Mode)
  •  The Standard Specific Requirement text is the associated requirement from the corresponding standard.
  • Questions Related to this Requirement (Displays in Requirements Mode)
  • The Questions Related to this Requirement text is the question(s) developed from the associated requirement from the corresponding standard.
  • Note: The Details/Standard section does not apply to all assessment types, therefore the button and section do not always display.
