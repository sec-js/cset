Diagramming Tool - File Menu

 
Selecting File in the Diagram Tool Bar displays the File dropdown menu.
 

From Template

  • Select the From Template option to open the Network Diagram Templates window.
  • Note: Selecting a new template overwrites any diagram components currently in the drawing area. In addition, any answers to questions related to those components will be removed from the assessment. To save the diagram without saving the associated answers, export the diagram.
Top

Save/Save As

  • Select the Save option to save the diagram.
  • Select the Save As option to name the diagram, select the file type, and save the file to a specified location.
Top

Share

  • Select the Share option to save and share the diagram on Google Drive.
Top

Import From

  • Select the Import From option to import a diagram file from one of the following locations:
  • Google Drive
  • OneDrive
  • DropBox
  • GitHub
  • GitLab
  • Browser
  • Device
  • URL
Top

Export As

  • Select the Export As option to export the diagram file as one of the following file types:
  • PNG
  • JPEG
  • SVG
  • PDF
  • VSDX (beta)
  • HTML
  • XML
  • URL
  • Select the Advanced option for additional export settings.
Top

Page Setup

  • Select the Page Setup option to adjust the paper size, orientation, grid side, and background.
Top

Print

  • Select the Print option to print the diagram.
  • Users can adjust the print settings as applicable and view a preview before printing.
Top

Close

  • Select the Close option to close the diagram.
  • Upon selection, and if the user has not saved the diagram, a confirmation dialog box displays the following message:
  • Select the Discard Changes button to close the diagram without saving.
  • Select the Cancel button to exit the window without closing the diagram.
