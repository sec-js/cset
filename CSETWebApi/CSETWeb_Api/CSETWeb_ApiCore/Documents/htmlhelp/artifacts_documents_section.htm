Artifacts/Documents Section

 
Selecting the Artifacts/Documents button on the assessment screen displays the Artifacts/Documents section.
 

Artifacts/Documents

  • The Artifacts/Documents section displays the following information for any previously added artifacts/documents:
  • Artifact/Document Title
  • Click the free text field to enter a title.
  • File Name
  • The name of the file uploaded to the assessment.
  • A red dot displays next to the Artifacts/Documents button when a question/requirement has an attached artifact or document.
Top

Add a Document

  • Select the Add a Document button to open the file explorer and import an external file.
Top

Related Questions

  • Select the Related Questions button to open the Related Questions dialog box.
  • The dialog box displays which question(s) or requirement(s) are associated with the artifact/document.
  • Select the OK button to close the dialog box.
Top

Delete

  • Select the delete button to remove the artifact/document.
  • A Please Confirm dialog box displays the following message:
  • Select the Yes button to remove the artifact/document.
  • Select the No button to keep the artifact/document and close the dialog box.
Top

Download

Select the download button to download the artifact/document.
