New Assessment

 
Select the New Assessment tab from the CSET main tabs to create a new assessment. The New Assessment tab displays a gallery of all standards and models in CSET.
 

Search

  • Enter a keyword into the Search bar to filter and search for an assessment.
  • Select the magnifying glass icon or press the return key to perform the search.
  • A filtered list of assessments will display in the assessment gallery based on the entered keyword.

Assessment Gallery

  • The assessment gallery organizes all CSET standards and models into the following categories:
  • Custom (if applicable)
  • Popular Assessments
  • CISA Sponsored (Resilience and Maturity)
  • Maturity Models
  • Energy and Electrical
  • Industrial and Utilities
  • Municipal and Health Care Services
  • NIST Special Publications
  • Financial CSET
  • Center for Internet Security (CIS) Assessments
  • Process Control and SCADA
  • Transportation
  • Other
  • DoDI and CNSSI

View Details

  • Select the View Details button to open the Assessment Description window which displays a brief overview of the assessment.
  • Select the X button to exit the Assessment Description window.
  • Select the triangle button to start the assessment.

Create Assessment

  • Select the Create Assessment button to start the assessment.

Navigation

  • Use the navigation buttons to scroll through the assessments in each category.
