Change Password

 
Users can select the Change Password option from the User Profile dropdown menu to display the Change Password window.
 

Password Requirements

  • The Change Password window displays a list of requirements for new passwords.

Password Fields

  • The Change Password window includes the following required fields:
  • Current Password
  • New Password
  • Confirm New Password
  • Select the Show Passwords button to reveal the text entered in each field.

Change Password

  • Select the Change Password button to save any changes and exit the Change Password window.

Cancel

  • Select the Cancel button to discard any changes and exit the Change Password window.
