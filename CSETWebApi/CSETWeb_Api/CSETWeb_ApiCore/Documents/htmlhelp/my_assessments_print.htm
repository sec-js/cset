My Assessments

 
Upon logging into CSET, the My Assessments tab is open by default. The My Assessments tab displays a list of all previously created assessments.
 

Import

  • Select the Import button to open the file explorer and choose an external CSET file.
  • Note: CSET only accepts .csetw file types.
  • Upon importing the file, the assessment will appear in the My Assessments tab.
  • If the assessment is encrypted, the CSET File Import window will open and the user will be required to enter a password before importing the assessment.

Turn off Export Encryption

  • Check the Turn Off Export Encryption check box to prevent the Encrypt Assessment window from displaying upon selecting the Export button.

My Assessments List

  • The My Assessments list displays the following information for any previously created assessments:
  • Assessment Name
  • The name of the assessment entered on the Assessment Configuration page.
  • Selected the hyperlinked assessment name to open the assessment.
  • Assessment Type
  • Indicates the name of the standard or model used for the corresponding assessment.
  • Last Modified
  • The date the assessment was last modified.
  • Primary Assessor
  • The first and last name of the primary assessor.
  • Status
  • Indicates the number of questions/requirements with responses within the assessment.
  • Note: For assessments that have a Questions Mode and Requirements Mode, the status will display the total number of answers for the mode that was last used within the corresponding assessment.
  • In addition, it will display a "Requires Review" message if the assessment contains a question that has been marked for review.
  • Select the Assessment Name, Assessment Type, Last Modified, or Primary Assessor column header to sort the listed assessments in ascending/descending order.
  • By default, assessments are ordered by date created.

Remove

  • Select the remove button to delete the assessment.
  • A Please Confirm dialog box displays the following message:
  • Select the Yes button to remove the assessment.
  • Select the No button to keep the assessment and close the dialog box.

Export

  • Select the Export button to download the assessment as a .csetw file.
  • Upon selecting the Export button, if the Turn Off Export Encryption checkbox is unchecked, the Encrypt Assessment window will display prior to downloading.
