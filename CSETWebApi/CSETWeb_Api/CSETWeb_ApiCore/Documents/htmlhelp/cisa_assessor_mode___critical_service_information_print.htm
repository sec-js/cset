CISA Assessor Mode - Critical Service Information

 
Selecting Critical Service Information within the Prepare menu displays the Critical Service Information page. This page is only available if CISA Assessor Workflow mode is enabled. All required fields must be completed to access the assessment's reports and exports.
 

Service Demographics

  • The Service Demographics section includes the following fields:
  • Critical Service Name
  • Enter the name of the critical service for which the assessment is to be performed.
  • Critical Service Description
  • Enter a description of the critical service for which the assessment is to be performed.
  • Name of IT/ICS System
  • Enter the name of the IT/ICS system for which the assessment is to be performed.
  • Multi-site Service
  • The No button is selected by default.
  • Select the Yes button if the service is performed at multiple sites.
  • Upon selection, the Multi-site Service Description free text field is enabled.
  • What is the basis of the cybersecurity budget for this Critical Service?
  • Select the option that best aligns with the basis of the cybersecurity budget for the critical service.
  • Total # of IT/ICS management and staff (i.e., technologists) assigned and/or dedicated to the Critical Service?
  • Select the appropriate numerical range.
  • Total # of IT/ICS cybersecurity management and staff (i.e., technologists) assigned and/or dedicated to the Critical Service?
  • Select the appropriate numerical range.
  • How many authorized, organizational users access and interact (i.e., operate, perform data entry, etc., as end users) with the Critical Service?
  • Select the appropriate numerical range.
  • How many authorized, non-organizational users access and interact (i.e., operate, perform data entry, etc., as end users) with the Critical Service?
  • Select the appropriate numerical range.
  • How many customers (i.e., individuals and/ or organizations) benefit from, depend on, and/or otherwise make use of the Critical Service?
  • Select the appropriate numerical range.

Service Composition

  • The Service Composition section includes the following fields:
  • Networks (wireless networks, wired networks, etc.)
  • Enter a description of the networks used for the critical service.
  • Services (computer services, e-mail servers, web servers, control services, etc.)
  • Enter a description of the services used for the critical service.
  • Applications (computer programs, ERP software, shareware user-added non-company software, etc.)
  • Enter a description of the applications used for the critical service.
  • Connections (VPN access by subcontractors, portable devices connected to organization services, interconnections between networks, connection of a Critical Service to the Internet, etc.)
  • Enter a description of the connections used for the critical service.
  • Individuals (key IT/OT security professionals, including administrators, users, and third-party vendors)
  • Enter a description of the individuals who contribute to the critical service.
  • Which of these cyber systems defines the Critical Service?
  • Select one primary cyber system that defines the critical service.
  • Once selected, all other checkboxes within the column are disabled.
  • The corresponding checkbox in the Secondary column is also disabled.
  • Select any secondary cyber systems that define the critical service.

Back
