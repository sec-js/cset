Component Question Overrides

 
Selecting an Override hyperlink within the Details/Standards section of a Diagram Component Question opens the Component Question Overrides window. This window allows users to enter a different answer to the default question for a specific component(s).
 
Users can select an answer button for the question, in which the answer applies to all component types listed in the Details/Standards section. However, if a component(s) requires a different answer, users can select the Override link for the component type.
 

Specific Components

  • Select the applicable answer button for each of the specified components.
  • The answer for the specific component(s) will take precedence over the answer selected for the overall default question.
  • Answers must be selected for all component names.
  • Note: If left blank, the answer will not display as the default question answer, but as "Unanswered" in the reports.

OK

  • Select the OK button to save any changes and exit the Component Question Overrides window.
  • Upon selecting the OK button, the corresponding default question will display under a new heading for that specific component with the overridden answer selected.
  • Users can update the answer/deselect the answer underneath the specific component header or within the Component Question Overrides window.
  • Note: If the answer is deselected, the question under the specific component header will not update to the default question answer, but will display as "Unanswered" in the reports.
