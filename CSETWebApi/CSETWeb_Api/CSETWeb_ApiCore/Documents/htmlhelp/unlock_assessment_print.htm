Unlock Assessment

 
Upon selecting the Enter Password button on the CSET File Import window, the Unlock Assessment window displays.
 

Password

  • Enter the assessment password into the Password field.
  • Note: Assessment passwords are created upon export on the Encrypt Assessment window.

Cancel

  • Select the Cancel button to cancel the import and exit the Unlock Assessment window.

Continue

  • Select the Continue button to submit the password.
  • If correct, CSET will import the assessment and the assessment will display in the My Assessments list.
  • If incorrect, an Invalid Password message will display on the CSET File Import window.
