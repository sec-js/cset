Diagram Component Questions

 
If the selected assessment contains a Network Diagram, the Diagram Component Questions screen will display in the Assessment section. Upon adding components to the network diagram, default questions associated with the components are added to the assessment.
 
For example, a clock component may have only one question in one category listed while an application server may have over forty questions across ten categories. A variety of component types will cause all the categories to display along with a large number of questions. Only questions that pertain to the components in the diagram will display. Questions that do not pertain to any components in the diagram will not be displayed or counted.
 
If a component is marked as having unique questions in the Properties tab of the Format Panel, the assessment will include a separate set of questions. Questions that were presented as component defaults will be asked again specifically for that component. A new heading for each component that was marked as having unique questions will be added to the assessment screen.
 
See Assessment Screen for additional information on how to navigate the features within the page.
