Reset Password

 
Existing users can click Reset Password on the Login page to receive a temporary password and reset the previous password.
 

Email

  • Enter the email address associated with the CSET account.

Next

  • If the email address provided is associated with a valid CSET account, the user will be prompted to answer a security question, if applicable.
  • Note: Users have the option to select security questions and answers during the account registration process.
  • Select Reset Password to have a temporary password emailed to the email address provided.

Login

  • Click Login to return to the Login page.
