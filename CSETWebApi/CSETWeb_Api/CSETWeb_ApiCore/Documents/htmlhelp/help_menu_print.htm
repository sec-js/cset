Help Menu

 
Selecting Help in the main navigation bar displays the Help dropdown menu. The Help menu allows users to access the user guide and other CSET documentation.
 

Help Menu

  • Select Help in the main navigation bar to display the Help dropdown menu which contains the following options:
  • CSET User Guide
  • Select the CSET User Guide option to open it in a new window.
  • About
  • Keyboard Shortcuts
  • Legal
  • Accessibility Statement
  • Advisory
  • Terms of Use
