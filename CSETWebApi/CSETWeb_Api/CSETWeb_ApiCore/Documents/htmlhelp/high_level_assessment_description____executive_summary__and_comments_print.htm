High-Level Assessment Description,
Executive Summary, and Comments

 
Selecting High-Level Assessment Description, Executive Summary and Comments within the Results menu displays the High-Level Assessment Description, Executive Summary, and Comments page. This page only displays based on the selected assessment.
 

High-Level Assessment Description

  • Enter a high-level description of the assessment to appear on reports.

Executive Summary

  • Enter an executive summary of the assessment to appear on reports.
  • An example summary is provided as default text.

Comments

  • Enter any general comments regarding the assessment to appear on reports.
