Question Listing

 
Selecting the Questions button on the Module Details page opens the Question Listing page where the user can add and manage the module questions.
 

Module Builder Navigation

  • Use the Module Builder Navigation links to access various pages throughout the Module Builder.
  • As the user progresses, additional page links will display.
Top

Add Question

  • Select the Add Question button to open the Add Question page.
Top

Question List

  • The Question List displays any previously created questions or questions cloned from a source module.
  • Users can update the Security Assurance Level (SAL) by selecting the applicable value(s).
  • See Security Assurance Level (SAL) for more information.
  • If available for the subcategory, users can click the pencil button to add a subcategory question.
Top

Edit

  • Select the Edit button to modify the question text.
  • Note: Not all questions are editable.
Top

Remove

  • Select the Remove button to delete the question from the module.
  • A Please Confirm dialog box displays the following message:
  • Select the Yes button to remove the question.
  • Select the No button to keep the question and close the dialog box.
Top

Back to Module Detail

  • Select the Back to Module Detail button to return to the Module Detail page.
