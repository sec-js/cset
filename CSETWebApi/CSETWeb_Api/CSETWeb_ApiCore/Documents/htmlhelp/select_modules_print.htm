Select Modules

 
Selecting the Clone from Existing Module(s) button on the Module Detail page opens the Select Modules window. Users can create modules from scratch or clone existing modules.
 

Existing Modules

  • Select one or more modules to clone by checking their corresponding checkbox.
  •  Note: Any current module content will be replaced by the requirements and questions from the selected modules.

Add Questions and Requirements

  • Select the Add Questions and Requirements button to clone the module(s).
  • Upon selection, the following Success message displays:
  • Select the OK button to acknowledge.
  • Upon selection, any questions and requirements associated with the cloned modules will display on the Requirement Listing and Question Listing pages.

Cancel

  • Select the Cancel button to exit the Select Modules window and return to the Module Detail page.
