Reports

 
Selecting Reports within the Results menu displays the Reports page.
 

Printing Reports Instructions

  • Select the Printing Reports Instructions button to open the printing instructions dialog box.
  • Select the Close button to exit the dialog box.

Reports

  • The Reports page displays a list of reports, worksheets, and checklists which varies based on the selected assessment. A brief overview is provided alongside each report.
  • Select the hyperlinked report name to open the report in a new tab.

Export

  • Select the Export button to download the assessment as a .csetw file.
