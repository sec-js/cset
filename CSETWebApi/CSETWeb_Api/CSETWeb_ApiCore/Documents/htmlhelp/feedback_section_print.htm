Feedback Section

 
Selecting the Feedback button on the assessment screen displays the Feedback section.
 

Feedback Section

  • Enter any feedback pertaining to the question/requirement.
  • Users can submit their feedback to the Cybersecurity and Infrastructure Security Agency (CISA) on the Feedback page under the Results section.
  • A red dot displays next to the Feedback button when a question/requirement has feedback.
