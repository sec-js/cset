Reference Document

 
Importing a new document or selecting a document's Edit button on the Standard Documents page opens the Reference Document page. Users can update the document's title, name, short name, summary, description and add comments.
 

Reference Document

  • Enter/edit the information in the following fields:
  • Title
  • Cannot be an existing title.
  • Name
  • Cannot be an existing name.
  • Short Name
  • Cannot be an existing short name.
  • Summary
  • Description
  • Comments
  • Note: This information is for the user's reference within the Module Builder and does not appear in the Resource Library.

Back to Reference Document List

  • Select the Back to Reference Document List button to save any changes and return to the Standard Documents page.
