Diagramming Tool - View Menu

 
Selecting View in the Diagram Tool Bar displays the View dropdown menu. A checkmark next to the option indicates the panel, window, or function is currently visible in the diagramming tool.
 

Format

  • Select the Format option to show/hide the Format Panel.

Outline

  • Select the Outline option to show/hide the Outline window.
  • For large and complex diagrams, use the Outline window to navigate the drawing area by clicking and dragging the blue square.
  • The area enclosed in the blue square indicates the diagram section currently displayed in the drawing area.

Layers

  • Select the Layers option to show/hide the Layers window.
  • Use layers to visually group sections of the diagram. Layers can be used to group backup systems, safety systems, redundant equipment, or components in isolated physical locations such as an offsite building, fenced area, substation, or remote pumping site.
  • The Layer window displays a list of all layers within the diagram.
  • The Main Layer is created by default.
  • Select the eye icon to show/hide the layer within the drawing area.
  • Note: Only visible layers will be included in the compliance analysis.
  • Select the lock/unlock icon to enable/disable the layer for editing.
  • If the diagram has more than one layer, select the arrows on the right to change the order of the layers.
  • If the layers overlap within the drawing area, the order within the Layers window determines which layer is in front/back of another layer.
  • Double click the layer to edit the layer name.
  • Select the plus icon to add another layer.
  • The layer will automatically display in the Layer window and will be named "Untitled Layer."
  • Select the duplicate icon to duplicate the layer.
  • The layer will automatically display in the Layer window and will be named "Untitled Layer."
  • The duplicate layer will include a copy of all objects within the original layer.
  • The duplicate objects will appear on top of the original objects within the drawing area.
  • Select the pencil icon to edit the label and add properties.
  • Enter a property name and select the Add Property button.
  • The property will be added as a field in which the user can add the property information.
  • Select the Export button to export the layer properties.
  • Select the Apply button to apply the properties to the layer.
  • Select the ellipses icon to move objects to another layer.
  • The ellipses icon is only enabled when an object(s) is selected within the drawing area.
  • Selecting the ellipses icon displays a list of the layers.
  • Select the layer in which to move the object(s).
  • A dot on the right side indicates the layer that currently contains the selected object(s).
  • Select the trash can icon to delete the selected layer.
  • If the deleted layer contains objects, the objects will be deleted.

Tags

  • Select the Tags option to show/hide the Tags window.
  • Use tags to associate labels with various objects.
  • Select the Add button to add a tag.
  • Enter the tag name and select the Add button.
  • The Tags window displays a list of all tags associated with the diagram.
  • Select the eye icon to show/hide any objects associated with the tag.
  • Selecting the checkbox highlights any objects in the drawing area associated with the tag.
  • The checkbox is only enabled if an object(s) is associated with the tag.
  • To associate an object with the tag:
  • Select the object.
  • Check the checkbox on the right side of the tag.
  • A blue checkbox indicates the object is associated with the tag.
  • Select the trash can icon to delete the tag.
  • Select the Reset button to unhide all tags and their associated objects.

Search Shapes/Scratchpad/Shapes

  • Select the Search Shapes option to show/hide the Search Shapes bar in the Symbols Panel.
  • Select the Scratchpad option to show/hide the Scratchpad section in the Symbols Panel.
  • Select the Shapes option to show/hide the shapes section in the Symbols Panel.
  • See Symbols Panel for more information.

Page View/Page Scale/Units

  • Select the Page View option to show the drawing area within the specified page size.
  • When Page View is not selected, the drawing area fills the entire window.
  • Select the Page Scale option to edit the scale of the page and adjust how small or large the diagram appears on the page.
  • Select the Units option to select one of the following units of measurement for the drawing area:
  • Points
  • Inches
  • Millimeters
  • Meters

Page Tabs/Tooltips/Ruler

  • Select the Page Tabs option to show/hide the page tabs.
  • Select the Tooltips option to show/hide any tooltips associated with any objects within the drawing area.
  • Select the Ruler option to show/hide the rulers alongside the drawing area.

Grids/Guides

  • Select the Grid option to show/hide the grid displayed in the background of the drawing area.
  • Select the Guides option to enable/disable guidelines within the drawing area.
  • When moving objects, guidelines automatically display whenever a central axis or object boundary aligns with those of another nearby object. This helps users align shapes relative to one another.

Connection Arrows/Connection Points

  • Select the Connection Arrows option to show/hide the connection arrows on an object.
  • When hovering over an object, connection arrows are four arrows that point from the top, bottom, left, and right of the object.
  • Selecting an arrow displays a symbols window.
  • Select a new object to connect to the existing object.
  • If an object already exist in the direction of the arrow, selecting the arrow with add a connection between the two objects.
  • Select the Connection Points option to show/hide the connection points on an object.
  • When hovering over an object, connection points are Xs that display around the object.
  • Select a connection point and drag the cursor to create a connection to another object.
  • See Adding Connectors for more information.

Reset View/Zoom In/Zoom Out

  • Select the Reset View option to reset any changes made to maximize or minimize the drawing area.
  • Select the Zoom In option to zoom into the drawing area.
  • Select the Zoom Out option to zoom out of the drawing area.

Fullscreen

  • Select the Fullscreen option to view the diagramming tool in fullscreen.
