Standard Documents

 
Selecting the Manage Documents button on the Module Detail or the Requirement Detail page opens the Standard Documents page. This page allows users to add, edit, and remove documents associated with the module.
 

Import a Document

  • Select the Import a Document button to open the file explorer and select a document to upload.
  • Upon selecting a document, the Reference Document page opens and the user can add additional document details.

Filter

  • Enter a keyword into the Filter bar to search for a document(s).
  • A filtered list of any documents that include the entered keyword will display in the Standard Documents list.
  • Delete the keyword text to remove the filter and view all documents.

Show Selected Items

  • Check/uncheck the Show Selected Items checkbox to show/hide any selected documents.

Standard Documents

  • The Standard Documents list displays all reference documents include with CSET, as well as any documents added by the user to support custom modules.
  • Note: For user-uploaded documents, enterprise users will have access to documents uploaded by others within the organization, whereas CSET Desktop users will only be able to view documents they have uploaded themselves.
  • The Standard Documents list includes the following fields:
  • Title
  • The title of the document.
  • File Name
  • The name of the file upon upload.
  • Check the checkbox next to any document to associate it with the module.
  • Upon selecting the Back button, the document(s) will appear in the Source Documents and Additional Documents dropdown menus in the References section of the Requirement Detail page.
  • Select the Edit button to open the Reference Document page.
  • Note: Not all documents within the Standards Documents list can be edited.

Back

  • Select the Back button to save any changes and return to the previous page.
