Overview

 
The following information provides a basic overview of the steps for conducting a CSET assessment.
 

Form the Assessment Team

Prior to beginning the assessment, select a cross-functional assessment team consisting of subject matter experts from various operational areas in the organization. Teams typically include representation from senior management, business, operations, IT, ICS, and security. Organizations may add additional team members as needed to address specific topics. The assembled team is responsible for determining the evaluation levels and answering specific, detailed questions on the control system and security configuration.
 
Familiarity with the tool will improve and speed up the assessment process. Anyone in the organization who has had training or experience with the tool should be included on the team. Alternately, the primary user should spend some time using the tool with test-only or dummy data prior to commencement of the team activity.
 
Representatives from the following areas are suggested for an effective assessment. The representatives should have significant expertise in their areas of responsibility.
 
If performing an ICS assessment:
  • Industrial Control Systems (knowledge of industrial control system architecture and operations)
  • System Configuration (knowledge of systems management)
  • System Operations (knowledge of system operation)
 
For either an ICS or IT assessment:
  • IT Network/Topology (knowledge of IT infrastructure)
  • IT Security/Control System Security (knowledge of policies, procedures, and technical implementation)
  • Risk Management (knowledge of the organization’s risk management processes and procedures)
  • Business (knowledge of budgetary issues and insurance postures)
  • Management (a senior executive sponsor/decision maker)
     

Gather Supporting Documentation and Information

Documents that may be referenced should be gathered prior to the assessment. Useful reference materials include information relating to operations, maintenance, physical security, cybersecurity, and hazardous materials.
 
Previous CSET users have found that the following types of documents and information are useful to have during completion of the assessment. Collecting this reference information before beginning the assessment is advisable:
 
  • Organizational chart that outlines responsibilities
  • Annual operating and capital budgets
  • Insurance policy description
  • Previously performed risk and vulnerability assessments
  • Capacity, operation, management, and maintenance manuals
  • Risk management documentation
  • Hazardous waste operations and emergency response standards
  • Emergency Operations Plan/Emergency Response Plan
  • Asset inventory and criticality rating from Computerized Maintenance Management System (CMMS)
  • Inventory list of process control/SCADA software and hardware, including interfaces
  • Network topology diagram and supporting documentation
  • Documentation/knowledge from previous incidents or near misses
  • General asset inventory, criticality asset determination, business impact analyses, contingency plans, etc.
  • Information security policies, plans, and procedures
 

Launch CSET

Users can install CSET using the following installation options:
 
CSET Desktop
CSET Desktop is a local installation of the software. Once installed, click the CSET icon to open the tool and begin an assessment. Visit https://github.com/cisagov/cset to download the latest release of CSET.
 
Enterprise Installation
To access the enterprise installation, contact your company's CSET administrator for the organization's unique CSET URL. After installation, navigate to the Login page and register a user account. Once logged in, the user can begin an assessment.
 

Select an Assessment Type

Once logged in, select a standard or model from the assessment gallery on the New Assessment tab. Each assessment contains a Prepare, Assessment, and Results section.
 

Prepare

The Prepare section includes several pages in which the user enters information such as organization details, contact information, and organization demographics. Depending on the selected assessment type, the tutorial, Security Assurance Level (SAL), and/or Network Diagram page may also display. It is recommended to read the tutorial in its entirety prior to conducting the assessment. Information entered in the Prepare section may impact the rigor and number of questions displayed in the Assessment section.
 

Conduct an Assessment

Within the Assessment section, the assessment screen includes a series of questions, requirements, or practices for which assessors provide responses. As a team, discuss and select the applicable answer button for each of the assessment questions. Unanswered questions are calculated as a 'No' response. For each question/requirement, users can add related artifacts/documents, observations, feedback, and comments. Responses provided on the assessment screen are compiled and presented in the Results section.
 
The selected assessment type, the assessment mode, the security assurance level (SAL), the components selected within a network diagram, and the information provided on the Prepare pages may impact the assessment structure and which questions/requirements/practices display on the assessment screen. These settings and selections help customize the assessment to better fit the organization, business unit, or critical service being assessed.
 

View and Analyze Results

Based on the selected standard or model, the Results section may display a variety of charts, tables, checklists, and reports summarizing the responses provided in the Assessment section.
 
The reports contain assessment details and results, which can assist the organization in its cybersecurity planning efforts. The reports help identify deficiencies or gaps in the organization's cybersecurity practices and suggest areas for improvement. Additionally, users can download and print the reports.
 

Additional Actions

Utilize Tools
Through the Tools dropdown menu, users have the ability to edit assessment parameters, export assessments to Excel, import modules, build modules, and perform trend and comparison aggregation functions.
 
Import/Export Assessments
Users can import and export CSET assessments as .cestw files.
 
Update User Language
Users can select the User Language option from the User Profile dropdown menu to display the User Language window. See User Language to view which languages and for which assessments are available.
 
Search the Resource Library
The Resource Library is a source for additional cybersecurity documentation, including standards and guidelines, and is accessed from the main navigation bar. See Resource Library for more information.
 
View Tutorials
The CSET YouTube page offers a variety of tutorials to help users navigate CSET and its features.

 

Submit Feedback
Selecting Feedback within the Results menu displays the Feedback page. This page allows users to submit any feedback regarding questions/requirements to the CSET Standards Administrator.
