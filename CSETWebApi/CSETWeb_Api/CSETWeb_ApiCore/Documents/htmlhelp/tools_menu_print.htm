Tools Menu

 
Selecting Tools in the main navigation bar displays the Tools dropdown menu. The Tools menu allows users to modify assessment features, import modules, create modules, aggregate assessment data, and more.
 

Tools Menu

  • Select Tools in the main navigation bar to display the Tools dropdown menu which contains the following options:
  • Enable Protected Features
  • Assessment Documents
  • Parameter Editor
  • Export Assessment to Excel
  • Import Modules
  • Module Builder
  • Module Content Report
  • Trend
  • Compare
  • Note: Assessment Documents, Parameter Editor, and Export Assessment to Excel are enabled only when the user is in an assessment.
