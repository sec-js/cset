Diagram Inventory

 
Selecting Network Diagram within the Prepare menu displays the Network Diagram page. This page is only available for the Network Diagram/Components Based Assessment. Once a user creates a network diagram, the Network Diagram page displays the diagram inventory.
 

Edit the Network Diagram

  • Select the Edit the Network Diagram button to open the diagramming tool and view the network diagram associated with the assessment.

Export to Excel

  • Select the Export to Excel button to export the diagram inventory information as an Excel file.

Diagram Inventory

  • The diagram inventory includes the following tabs:
  • Components
  • Zones
  • Shapes
  • Text
  • Links
  • Network Warnings
  • Select each tab to view the corresponding information for each object type.
