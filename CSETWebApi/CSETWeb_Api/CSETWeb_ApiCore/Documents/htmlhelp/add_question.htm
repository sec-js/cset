Add Question

 
Selecting the Add Question button on the Question Listing page opens the Add question page. Users have the option to create a new question or search and associate an existing question with the module.
 

Write New Question

  • To write a new question, add the question text in the free text field.
  • Users can find previously created questions in the Search for Existing Questions section.
Top

Categorization

  • Select the applicable Question Group Heading from the dropdown menu.
  • Select the applicable Subcategory from the dropdown menu or enter free text.
Top

Security Assurance Level

  • Select the applicable Security Assurance Levels for the question.
  • See Security Assurance Level (SAL) for more information.
Top

Add New Question

  • Select the Add New Question button to create the question and associate it with the module.
  • Upon selection, the user is returned to the Question Listing page.
Top

Search for Existing Questions

  • Enter keyword(s) related to the desired question and select the Search button.
  • Upon selection, a list of any relevant questions will display below the search bar.
  • For the desired questions, select the applicable SALs and select the Plus button.
  • Note: The SALs for each question are automatically selected in accordance to how they were defined in their original module. Users have the option to modify the SALs as appropriate for the new module.
  • Once all questions are chosen, select the Add Selected Questions button.
  • Upon selection, the user is returned to the Question Listing page.
  • Note: The Add Selected Questions button is only enabled once one or more questions have been chosen.
Top

Back to Question List

  • Select the Back to Question List button to return to the Question Listing page.
