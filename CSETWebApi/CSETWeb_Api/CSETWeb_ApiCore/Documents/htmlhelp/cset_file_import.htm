CSET File Import

 
Upon selecting Import or Import an Existing Assessment, the CSET File Import window displays. If the assessment is encrypted, the CSET File Import window will prompt the user to enter a password. If the assessment is not encrypted, the window will automatically close after completing the upload.
 

Uploading Files

  • The CSET File Import window displays the following information:
  • The assessment file name
  • The password hint (if encrypted)
Top

Cancel

  • Select the Cancel button to cancel the import and exit the CSET File Import window.
Top

Enter Password

  • The Enter Password button only displays if the assessment is encrypted.
  • Select the Enter Password button to open the Unlock Assessment window and enter the assessment password.
