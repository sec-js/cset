Supplemental Guidance Section

 
Selecting the Supplemental Guidance button on the assessment screen displays the Supplemental Guidance section. The Supplemental Guidance button may contain either an "i" or "G" icon.
 

Supplemental Guidance

  • The Supplemental Guidance section provides the user with additional information regarding the associated question or requirement.
  • Since each assessment type and their questions/requirements are unique, the information provided varies.
