Diagramming Tool - Help Menu

 
Selecting Help in the Diagram Tool Bar displays the Help dropdown menu.
 

Search

  • Use the Search field to find additional information on features within the diagramming tool.
  • Enter a keyword or topic into the search bar and press the Enter key.
  • Upon selecting Enter, a list of relevant help topics will open in the user's browser.

Keyboard Shortcuts

  • Select the Keyboard Shortcuts option to open a list of diagramming tool shortcuts in the user's browser.

Quick Start Video

  • Select the Quick Start Video option to open a diagramming tool introductory video in the user's browser.

Support

  • Select the Support option to open the diagramming tool's GitHub support page in the user's browser.

About Version

  • Select the About Version option to open information about the diagramming tool in the user's browser.
