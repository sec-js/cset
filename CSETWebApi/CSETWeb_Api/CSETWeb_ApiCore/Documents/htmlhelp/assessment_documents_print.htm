Assessment Documents

 
Users can select the Assessment Documents option from the Tools dropdown menu to display the Assessment Documents window. The Assessments Documents menu option is only enabled when the user is in an assessment.
 
The Assessment Documents window displays a list of all documents added to the assessment via the Artifacts/Documents section during the assessment. These files are associated with specific practices to help explain or to provide evidence for the answer given.
 

Assessment Documents

  • The Assessment Documents window displays the following information for any previously added documents:
  • Document Title
  • The title entered by the user during the assessment via the Artifacts/Documents section.
  • File Name
  • The name of the file uploaded to the assessment via the Artifacts/Documents section.

Download

  • Select the Download button to download all documentation associated with the assessment.

OK

  • Select the OK button to exit the Assessment Documents window.
