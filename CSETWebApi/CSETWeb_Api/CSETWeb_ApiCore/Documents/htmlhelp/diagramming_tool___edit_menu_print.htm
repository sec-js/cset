Diagramming Tool - Edit Menu

 
Selecting Edit in the Diagram Tool Bar displays the Edit dropdown menu.
 

Undo/Redo

  • Select the Undo option to reverse the last action.
  • Select the Redo option to reverse the last undo action.
  • The Redo option is only enabled after undoing an action.
  • Note: The Undo and Redo functions have limitations in terms of the number and types of actions performed.

Cut/Copy/Paste/Delete

  • Select the Cut option to remove the selected object(s).
  • The cut object will be added to the clipboard and can be pasted onto the drawing area.
  • Select the Copy option to copy the selected object(s).
  • The copied object will be added to the clipboard and can be pasted onto the drawing area.
  • Select the Paste option to paste the cut or copied object(s) from the clipboard onto the drawing area.
  • Note: Non-CSET objects cannot be pasted into the diagramming tool.
  • Select the Delete option to remove the selected object(s).
  • Note: Cutting and deleting an object will permanently remove any associated questions and answers from the assessment. If the object is re-added or pasted, any previous answers associated with the object will not be re-added to the assessment.

Duplicate

  • Select the Duplicate option to copy and paste the selected object(s) onto the drawing area.

Find/Replace

  • Select the Find/Replace option to search the drawing area for specific text and replace it with the entered text.
  • The search will find any objects with a label or metadata matching the entered text.
  • Selecting the Regular Expression checkbox allows users to search for regular expression in the JavaScript syntax.

Edit

  • Select the Edit Data option to edit the object's data.
  • Different fields display based on the object type.
  • Select the Edit Tooltip option to add or edit a tooltip associated with the selected object.
  • The tooltip displays when a user hovers over the corresponding object.
  • Select the Edit Style option to edit the style of the selected object.
  • Select the Edit Geometry option to edit the shape of the selected object.
  • Select the Edit option to edit the label text of the selected object.
  • Select the Open Link option to open the link associated with the selected object.
  • The Open Link option is only enabled when the selected object is associated with a link.

Select

  • Select the Select Vertices option to select all symbol vertices within the diagram.
  • Select the Select Edges option to select all connectors within the diagram.
  • Select the Select All option to select all objects within the diagram.
  • Select the Select None option to deselect all objects within the diagram.

Lock/Unlock

  • Select the Lock/Unlock option to:
  • Lock the selected object(s) and prevent them from being moved or edited.
  • Unlock the selected object(s) and enable them to be moved and edited.
