Diagramming Tool - Arrange Menu

 
Selecting Arrange in the Diagram Tool Bar displays the Arrange dropdown menu.
 

To Front/To Back

  • Select the To Front option to bring the selected object in front of all other objects.
  • The To Front option is only enabled if an object is selected.
  • Select the To Back option to send the selected object behind all other objects.
  • For example: Sending a zone to display behind the components.
  • The To Back option is only enabled if an object is selected.

Bring Forward/Send Backward

  • Select the Bring Forward option to bring the selected object in front of the object one position in front of it.
  • The Bring Forward option is only enabled if an object is selected.
  • Select the Send Backward option to send the selected object behind the object one position behind it.
  • The Send Backward option is only enabled if an object is selected.

Direction/Rotate

  • Select the Direction option to adjust the direction of the selected object.
  • The Direction option is only enabled if an object is selected.
  • From the Direction dropdown menu, users can select:
  • Flip Horizontal
  • Flip Vertical
  • Rotation
  • Selecting the Rotation option opens the rotation dialog box.
  • Enter the degree in which to rotate the selected object.
  • Select the Apply button to apply the changes.
  • Select the Cancel button to discard any changes and exit the dialog box.
  • Select the Rotate Shape only by 90° / Reverse option to rotate the selected object by 90° degrees.
  • Selecting the option again will reverse the action and return the shape to its previous direction.
  • The Rotate Shape only by 90° / Reverse option is only enabled if an object is selected.

Align/Distribute

  • Select the Align option to align a group of objects with one another.
  • The Align dropdown menu options are only enabled if more that one object is selected.
  • From the Align dropdown menu, users can select:
  • Left Align
  • Aligns all objects to the left side of the furthest object on the left.
  • Center
  • Vertically aligns all objects to the center point of the first selected object.
  • Right Align
  • Aligns all objects to the right side of the furthest object on the right.
  • Top Align
  • Aligns all objects to the top side of the highest object.
  • Middle
  • Horizontally aligns all objects to the middle point of the first selected object.
  • Bottom Align
  • Aligns all objects to the bottom side of the lowest object.
  • Snap to Grid
  • Assists the user in aligning selected objects to the grid.
  • Select the Distribute option to equally distribute the space between the selected objects.
  • The Distribute dropdown menu options are only enabled if more than one object is selected.
  • From the Distribute dropdown menu, users can select:
  • Horizontal
  • Evenly distributes the space between the objects horizontally.
  • Vertical
  • Evenly distributes the space between the objects vertically.

Navigation

  • Select the Navigation option to navigate groups and zones.
  • The Navigation dropdown menu options are only enabled if a zone is selected.
  • From the Navigation dropdown menu, users can select:
  • Home
  • Select the Home option to return to the diagram.
  • Exit Group
  • Select the Exit Group option to exit the group or zone and view the entire diagram.
  • Enter Group
  • Select the Enter Group option to enter the group or zone and view only the objects within the group or zone.
  • Expand
  • Select the Expand option to expand the zone.
  • Collapse
  • Select the Collapse option to collapse the zone.
  • Collapsible
  • Select the Collapsible option to show/hide the zone's expand/collapse button in the top left corner.

Insert

  • Select the Insert option to insert an object.
  • From the Insert dropdown menu, users can select:
  • Rectangle
  • Ellipse
  • Rhombus
  • Line
  • Note
  • Text
  • Link
  • Shape
  • Selecting the Shape option opens the Edit Shape window.
  • Select the Help button for information on how to use this feature.
  • Note: This feature is intended for developer use.
  • Freehand
  • Image
  • Template
  • Layout
  • From the Layout dropdown menu, users can select:
  • Horizontal Flow
  • Select the Horizontal Flow option to create a diagram that flows horizontally.
  • Vertical Flow
  • Select the Vertical Flow option to create a diagram that flows vertically.
  • Horizontal Tree
  • Select the Horizontal Tree option to create a horizontal tree diagram.
  • Vertical Tree
  • Select the Vertical Tree option to create a vertical tree diagram.
  • Radial Tree
  • Select the Radial Tree option to create a radial tree diagram.
  • Organic
  • Select the Organic option to create an organic diagram.
  • Circle
  • Select the Circle option to create a circular diagram.
  • Advanced
  • From the Advanced dropdown menu, users can add custom diagram layouts using:
  • Text
  • Mermaid syntax
  • SQL code
  • CSV data

Layout

  • Select the Layout option to modify the diagram layouts.
  • From the Layout dropdown menu, users can select one of the following layouts in which to update all diagrams within the drawing area:
  • Horizontal Flow
  • Vertical Flow
  • Horizontal Tree
  • Vertical Tree
  • Radial Tree
  • Organic
  • Circle
  • Org Chart
  • Select the Parallels option to expose and separate overlapping connectors between shapes.
  • Select the Custom option to customize a layout.

Group

  • Select the Group option to group selected objects together.
  • The Group option is only enabled when an object(s) is selected.
  • Once in a group, all objects will move together.
  • Select the Ungroup option to remove the previously grouped objects from the group.
  • The Ungroup option is only enabled when a grouped object(s) is selected.
  • Once ungrouped, the objects will move individually.
  • Select the Remove from Group option to remove a specific object from an existing group.
  • The Remove from Group option is only enabled when a grouped object(s) is selected.

Clear Waypoints/Autosize

  • Select the Clear Waypoints option to remove all or just the selected waypoints from the diagram.
  • See Adding Connectors for more information.
  • Select the Autosize option to to adjust all selected objects to the height of their text labels.
