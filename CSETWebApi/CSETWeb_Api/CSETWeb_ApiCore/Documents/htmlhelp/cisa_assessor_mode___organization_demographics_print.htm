CISA Assessor Mode - Organization Demographics

 
Selecting Organization Demographics within the Prepare menu displays the Organization Demographics page. This page is only available if CISA Assessor Workflow mode is enabled. All required fields must be completed to access the assessment's reports and exports.
 

Import/Export

  • Select the Import button to import a JSON file of the organization's demographics.
  • Select the Export button to export the organization's demographics as a JSON file.
  • Note: This is helpful when assessors are conducting a new assessment for an organization that has already undergone an assessment, and its demographic information has been previously entered.

Demographics

  • The demographics section includes the following fields:
  • Organization Type
  • Select the organization type from the dropdown menu that best aligns with the organization for which the assessment is to be performed.
  • Note: The selected organization type may affect the terminology used in the following questions.
  • Name of Organization
  • Enter the name of the organization for which the assessment is to be performed.
  • Business Unit
  • Enter the name of the business unit or agency within the organization for which the assessment is to be performed.
  • Sector
  • Select the sector from the dropdown menu that best aligns with the organization for which the assessment is to be performed.
  • Sub-Sector
  • Select the sub-sector from the dropdown menu that best aligns with the organization for which the assessment is to be performed.
  • Note: A sector must be selected for sub-sector options to appear in the dropdown menu. The options available within the dropdown menu are based on the selected sector.
  • Number of Employees in the Government Agency/Organization
  • Select the numerical range from the dropdown menu that best aligns with the agency/organization for which the assessment is to be performed.
  • Number of Employees in the Department/Business Unit
  • Select the numerical range from the dropdown menu that best aligns with the department/business for which the assessment is to be performed.
  • Total Annual Funding for Governmental Organization or Agency/Total Annual Revenue for Organization
  • Select the dollar amount range from the dropdown menu that best aligns with the organization/agency for which the assessment is to be performed.

Standards and Regulations

  • The standards and regulations section includes the following fields:
  •  Does your organization use a body of practice, industry standard, or similar resource to support or inform its cybersecurity efforts?
  • The Yes button is selected by default.
  • Enter the two most important bodies of practice, industry standards, or similar resources used.
  • Select the No button if the organization does not use a body of practice, industry standard, or similar resource to support or inform its cybersecurity efforts.
  • Upon selection, the Most Important and Second Most Important text fields are hidden.
  • Is your organization required to comply with mandatory, cybersecurity-related regulation?
  • The Yes button is selected by default.
  • Select the two primary regulation types from the listed choices or enter them in the free text fields.
  • Select the No button if the organization is not required to comply with mandatory, cybersecurity-related regulation.
  • Upon selection, the regulation type choices and free text fields are hidden.
  • Please indicate the group(s) that your organization shares with or obtains cybersecurity-related information from.
  • Check all listed options that apply and enter any additional groups in the free text field.

Significant Barriers

  • Enter the most significant and second most significant barriers or concerns limiting the organization's participation in cybersecurity information sharing groups or initiatives.

Back

  • Select the Back button to return to the previous page.
