Add Requirement

 
Selecting the Create Requirement button on the Requirement Listing page opens the Add Requirement window where the user can add a new requirement.
 

Requirement Fields

  • The Requirement section includes the following required fields:
  • Category
  • Enter a category using free text or select a category from the dropdown menu.
  • To search the dropdown menu, users can enter text to filter the options.
  • Question Group Heading
  • Enter the name of the heading that questions associated with the requirement are grouped under when displayed in questions mode.
  • Subcategory
  • Enter a subcategory using free text or select a subcategory from the dropdown menu.
  • To search the dropdown menu, users can enter text to filter the options.
  • Title/Identifier
  • Enter the title or identifier of the requirement.
  • Requirement Text
  • Enter the text of the requirement.
  • Note: Line breaks are preserved and display in the assessment.
