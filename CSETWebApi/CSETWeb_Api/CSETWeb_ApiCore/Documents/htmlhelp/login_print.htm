Login

 
Prior to accessing CSET, users are directed to the Login page. Users are required to enter login credentials before utilizing the tool. CSET Desktop users will be logged in automatically.
 

Login Credentials

  • If an existing user, enter the following login credentials:
  • Email
  • Password

Login

  • After entering the login credentials, select the Login button to access CSET.

Reset Password

  • If an existing user, click Reset Password to receive a temporary password and reset the previous password.

Register New Account

  • If a new user, click Register a New Account to create a user profile.
