User Profile Menu

 
Selecting the user profile in the main navigation bar displays the User Profile dropdown menu. The User Profile menu allows users to view their user profile information and assessments, change their password, select a user language, and log out of CSET.
 

User Profile Menu

  • Select the user profile in the main navigation bar to display the User Profile dropdown menu which contains the following options:
  • User Profile
  • My Assessments
  • Change Password
  • User Language
  • Logout
  • Select the Logout option to exit CSET and return to the Login page.
  • Note: When using CSET Desktop, the only option available in the User Profile menu is "My Assessments."
