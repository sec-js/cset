Resource Library - Search

 
Selecting Resource Library in the main navigation bar displays the Resource Library window with the Search tab displaying by default. The Search tab option of the Resource Library provides a way to find a list of documents based on the text string typed into the search box.
 
The figure below shows an example where the user has typed in "CPG." In this case, CSET searches through all the documents for occurrences of "CPG" and then displays the documents in the Search Results.
 

Search Field

  • Use the Search field to enter one or more keywords related to the desired documents, such as the topic, the authoring organization, or document type.
  • CSET performs searches within documents titles and content. CSET does not search for similar spellings or misspellings of entered keywords.
  • Users can use the following wildcard characters to perform a search:
  • Asterisk (*)
  • Use an asterisk to substitute one or more characters.
  • For example: Searching for the text "fire*" would return any document with any word starting with those characters. In this case, the search results would prioritize topics related to firewalls first. Without the asterisk, the search results would display topics related to fire protection first.
  • Question Mark (?)
  • Use question marks to substitute exact characters.
  • For example: Searching for the text "NIST SP800-??" would return any document that includes any two additional characters after the "NIST SP800-" text. In this case, the search results would prioritize the NIST Special Publication 800 series documents.

Search

  • After entering keywords into the Search field, select the Search button to perform the search.

Search Results

  • Upon selecting the Search button, any documents that include the entered keyword(s) will display in the Search Results list.
  • Users can click the hyperlinked document title to open the document.
