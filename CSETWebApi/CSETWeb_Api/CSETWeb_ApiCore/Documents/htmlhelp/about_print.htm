About

 
Users can select the About option from the Help dropdown menu to display the About window. The About window provide information about the CSET tool, team, and resources.
 

About CSET

  • The About window includes the following information about CSET:
  • CSET version
  • The CSET Version text indicates which version of CSET is currently installed.
  • Note: Referencing the version may be helpful when communicating questions or concerns to the CSET support team.
  • CSET website
  • The CSET GitHub allows users to learn more about the CSET tool, download new releases, request features, and report defects or problems.
  • Contact information
  • Training resources
  • Training program

OK

  • Select the OK button to exit the About window.
