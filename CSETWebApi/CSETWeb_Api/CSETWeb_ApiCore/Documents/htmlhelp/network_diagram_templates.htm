Network Diagram Templates

 
Selecting the Create a New Diagram button on the Network Diagram page opens the diagramming tool and the network diagram templates window. Users can also select the From Template option within the File dropdown menu to open the window. When starting a network diagram, users can create from scratch or select a template to customize.
 

File Name

  • Enter the name of the template into the Fillename field.
Top

File Type

  • Select the file type for the template from the dropdown menu.
  • Select the help icon for more information about file types.
Top

Diagram Templates

  • Users can select a pre-built network diagram from the templates window.
Top

Help

  • Select the Help button for more information about using the diagramming tool.
Top

Cancel

  • Select the Cancel button to exit the network diagram templates window and create a new network diagram from scratch.
Top

From Template URL

  • Select the From Template URL button to import a network diagram template.
Top

Create

  • After selecting a network diagram template, select the Create button to open the template in the diagramming tool.
