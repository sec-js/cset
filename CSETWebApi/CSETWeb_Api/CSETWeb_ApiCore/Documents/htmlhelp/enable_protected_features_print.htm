Enable Protected Features

 
Users can select the Enable Protected Features option from the Tools dropdown menu to display the Protected Features window. This window allows the user to enable modules and CISA Assessor Workflow mode.
 

Enable Modules

  • Select the Enable Modules button to enable the modules listed above.
  • Note: Once selected, the modules will remain enabled for the user.

CISA Assessor Workflow

  • Select the On button to enable CISA Assessor Workflow.
  • CISA Assessor Workflow is turned off by default.
  • The toggle option highlighted in blue indicates the currently selected option.
  • When enabled, the user will see the following screens in the Prepare section at the beginning of each assessment:
  • Organization Demographics
  • Assessment Configuration
  • Critical Service Information
  • Note: When CISA Assessor Workflow is enabled, completion of all Organization Demographics, Assessment Configuration, and Critical Service Information fields are required to access assessment reports and exports.

Close

  • Select the Close button to save any changes and exit the Protected Features window.
