Assessment Information

 
Selecting Assessment Information within the Prepare menu displays the Assessment Information page.
 

Contacts

  • The Contacts section displays a list of personnel associated with the assessment.
  • Users can add contacts by selecting the Add Contacts button.
  • Once added, the contact card will display.
  • Select the Change button to edit the contact card.
  • Select the Email button to open the Invitation Email window to invite the user to participate in the assessment.
  • Select the Remove button to delete the contact card.
  • A Please Confirm dialog box displays the following message:
  • Select the Yes button to remove the contact.
  • Select the No button to keep the contact and close the dialog box.
  • The user who created the assessment is automatically listed as the Assessment Owner.
  • This contact information cannot be edited.

Add Contacts

  • Select the Add Contacts button to associate additional personnel with the assessment.
  • Upon selection, the Add Contacts form displays.
  • Enter the applicable information for each field.
  • Select one of the following roles that best applies to the contact:
  • Participant
  • Participates in the assessment but cannot add additional contacts.
  • Facilitator
  • Facilitates the assessment and can add additional contacts.
  • Select the Save button to add the contact.
  • Select the Cancel button to disregard any changes and close the Add Contacts form.

Import/Export

  • Select the Import button to import a JSON file of the organization's demographics.
  • Select the Export button to export the organization's demographics as a JSON file.
  • Note: This is helpful when assessors are conducting a new assessment for an organization that has already undergone an assessment, and its demographic information has been previously entered.

Demographics

  • The Demographics section includes the following fields:
  • Sector
  • Select the sector from the dropdown list that best aligns with the organization for which the assessment is to be performed.
  • Industry
  • Once the user selects a sector, the Industry dropdown field will populate with options relevant to the selected sector.
  • Select the industry from the dropdown list that best aligns with the organization for which the assessment is to be performed.
  • Asset Gross Value
  • Select a dollar value estimate of the organization's assets from the dropdown list.
  • Relative Effort
  • Select the estimated amount of time it will take to complete the assessment from the dropdown list.
  • Name of Organization
  • Enter the name of the organization for which the assessment is to be performed.
  • Business Unit/Agency
  • Enter the name of the business unit or agency within the organization for which the assessment is to be performed.
  • Organization Type
  • Select the organization type from the dropdown list that best aligns with the organization for which the assessment is to be performed.
  • Organization Point of Contact
  • The Organization Point of Contact dropdown list includes all contacts associated with the assessment.
  • Select the point of contact for the organization.
  • Facilitator
  • The Facilitator dropdown list includes all contacts associated with the assessment.
  • Select the facilitator who will be performing the assessment from the dropdown list.
  • Critical Service Name
  • Enter the name of the critical service for which the assessment is to be performed.
  • Critical Service Point of Contact
  • The Critical Service Point of Contact dropdown list includes all contacts associated with the assessment.
  • Select the point of contact for the critical service.

Back

  • Select the Back button to return to the previous page.

Next

  • Select the Next button to go to the following page.
