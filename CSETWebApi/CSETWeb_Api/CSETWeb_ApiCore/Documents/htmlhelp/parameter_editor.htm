Parameter Editor

 
Users can select the Parameter Editor option from the Tools dropdown menu to display the Parameter Editor window. The Parameter Editor menu option is only enabled when the user is in an assessment.
 
Some standards in CSET contain parameter information within the requirement text. Parameter information is displayed as blue text enclosed in brackets. If the standard contains parameter information, the Parameter Editor will display a list of the associated parameters.
 

Parameter List

  • The parameter list displays the following information:
  • Parameter Name
  •  A read-only field of the name of the parameter.
  • Default Parameter Value
  • Initially the same as the Parameter Name, it's the parameter text that currently displays within the requirement text for the selected standard.
  • Select the blue Default Parameter Value to replace it with user-defined text.
  • For example: The user can replace the [Assignment: organization-defined frequency] parameter with the word "annually." Within the standard, all occurrences of the parameter will be updated with the user-defined text.
  • Note: Users can also change parameter text within the requirement text itself. See requirements for more information.
