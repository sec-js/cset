Adding a Multiple Services Component (MSC)

 
When multiple symbols are needed to represent a single physical device, such as a firewall and IDS, the Multiple Services Component (MSC) may be used to group the symbols. Like zones, the MSC is essentially a boundary or border in which to enclose the components.
 

MSC Symbol

  • Click the Multiple Service Component symbol from the Symbols Panel and drag it to the drawing area.

Multiple Service Component

  • Add symbols to the MSC by clicking and dragging them onto the MSC area.
  • Users can add connectors between the symbols and other zones.
  • Note: In the assessment analysis and reports, the MSC will not be shown. Instead, each included service will be listed individually. The default tags include the MSC designation and then the service abbreviation.
  • Example: An application server within Multiple Services Component 1 (MSC-1) will be labeled MSC-1-AS-x where as "x" is a sequential number. The default tag can be changed by double-clicking in the tag field.
