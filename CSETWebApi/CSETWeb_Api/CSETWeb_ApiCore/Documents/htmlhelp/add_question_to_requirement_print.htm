Add Question to Requirement

 
Selecting the Add Question button on the Requirement Detail page opens the Add Question to Requirement page. Users have the option to associate a new question or search and associate an existing question with the requirement.
 

Requirement Details

  • The requirement details section displays the requirement title and requirement text.

Write New Question

  • To write a new question, add the question text in the free text field.
  • Users can find previously created questions in the Search for Existing Questions section.

Categorization

  • Select the applicable Question Group Heading from the dropdown menu.
  • Select the applicable Subcategory from the dropdown menu or enter free text.

Security Assurance Level

  • Select the applicable Security Assurance Levels for the question.
  • See Security Assurance Level (SAL) for more information.

Add New Question

  • Select the Add New Question button to create the question and associate it with the requirement.
  • Upon selection, the user is returned to the Requirement Detail page with the question listed under the Related Questions section.

Search for Existing Questions

  • Enter keyword(s) related to the desired question and select the Search button.
  • Upon selection, a list of any relevant questions will display below the search bar.
  • For the desired questions, select the applicable SALs and select the Plus button.
  • Note: The SALs for each question are automatically selected in accordance to how they were defined in their original module. Users have the option to modify the SALs as appropriate for the new requirement.
  • Once all questions are chosen, select the Add Selected Questions button.
  • Upon selection, the user is returned to the Requirement Detail page with the question(s) listed under the Related Questions section.
  • Note: The Add Selected Questions button is only enabled once one or more questions have been chosen.

Back to Requirement

  • Select the Back to Requirement button to return the the Requirement Detail page.
