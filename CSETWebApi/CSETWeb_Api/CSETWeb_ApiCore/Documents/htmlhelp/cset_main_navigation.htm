CSET Main Navigation

 
Upon logging into CSET, users are directed to the My Assessments tab by default. Users can navigate CSET through the main navigation bar and tabs.
 

CSET Main Menu

  • The CSET main menu includes the following options:
  • File
  • Edit
  • View
  • Window
  • These menu options allow the user to print, cut, copy, paste, adjust the CSET window, view developer tools, and more.
  • Note: The CSET main menu only displays for CSET Desktop users.

CSET Main Navigation Bar

  • The CSET main navigation bar includes the following options:
  • CSET Home
  • Select the CSET home button to return to the My Assessments tab.
  • Tools Menu
  • Resource Library
  • Help Menu
  • User Profile
Top

CSET Main Tabs

  • The CSET main tabs include the following options:
  • New Assessment
  • My Assessments
Top

Start a New Assessment

  • Select the Start a New Assessment button to open the New Assessment tab.
  • Note: The Start a New Assessment button only displays if the user does not have any existing assessments.
Top

Import an Existing Assessment

  • Select the Import an Existing Assessment button to open the file explorer and choose an external CSET file.
  • Note: CSET only accepts .csetw file types.
  • Upon importing the file, the assessment will appear in the My Assessments tab.
  • If the assessment is encrypted, the CSET File Import window will open and the user will be required to enter a password before importing the assessment.
