Comparison Details

 
Selecting the New Comparison button from the Comparisons window opens the Comparison Details window.
 

Comparison Name

  • Enter a comparison name in the required Comparison Name field.
Top

Select Assessments

  • Select the Select Assessments button to open the Select Assessments window and choose which assessments to analyze.
Top

Selected Assessments List

  • Assessments chosen on the Select Assessments window will display in the Selected Assessments List.
  • The Selected Assessments List displays the following information:
  • Alias
  • Enter a unique alias for each assessment to distinguish the results between each assessment on the Compare Assessments page.
  • The user can change the pre-populated name by clicking within the free text field.
  • Assessment Name
  • The name of the assessment entered on the Assessment Configuration page.
  • Assessment Date
  • The date the assessment was created.
  • Assessment Type Columns
  • Indicates the name of the standard or model used for the corresponding assessment.
