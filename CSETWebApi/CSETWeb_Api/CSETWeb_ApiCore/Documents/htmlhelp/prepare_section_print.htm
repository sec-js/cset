Prepare Section

 
Upon selecting a new assessment or a previously created assessment, the assessment opens and displays the Prepare tab by default. Users can select the Prepare tab or select a sub-page under the Prepare header within the side navigation to view the Prepare section. This section allows users to enter information about the organization, sector, assessment facilitator and participant contact information, critical service information, and more.
 

Prepare Menu

  • The following sub-menu pages may display based on the selected assessment:
  • Tutorial
  • Some assessments include tutorials that provide an overview of the assessment's purpose, scope, structure, components, and any additional resources.
  • Assessment Configuration
  • Assessment Information
  • Security Assurance Level (SALs)
  • Network Diagram
  • If CISA Assessor Workflow mode is enabled, the following pages will display:
  • Organization Demographics
  • Assessment Configuration
  • Critical Service Information
