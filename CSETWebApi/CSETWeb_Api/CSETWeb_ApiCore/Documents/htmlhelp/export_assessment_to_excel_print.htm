Export Assessment to Excel

 
Users can select the Export Assessment to Excel option from the Tools dropdown menu to display the Excel Export window. The Export Assessment to Excel menu option is only enabled when the user is in an assessment.
 

Export

  • Select the Export button to export the assessment results as an excel file.
  • Note: Excel data cannot be reimported. The excel file will include either questions or requirements, whichever mode had more answers within the assessment.

Cancel

  • Select the Cancel button to exit the Excel Export window.
